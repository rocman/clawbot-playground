# 🏰 Tower Defence

A fully-featured H5 tower defence game built with pure HTML5 Canvas + Vanilla JS. No dependencies, no build step — just open `index.html`.

## 🎮 How to Play

1. Select a tower type from the side panel
2. Click any **green (grass) tile** on the map to build
3. Hit **「发起进攻」** to start the next wave
4. Survive all **15 waves** to win!

## 🗼 Tower Types

| Tower | Cost | Specialty |
|-------|------|-----------|
| 🏹 弓箭手 (Archer) | 50 | Fast, reliable single-target |
| 💣 火炮 (Cannon)  | 100 | AOE splash damage |
| ❄️ 冰塔 (Ice)     | 80  | Slows enemies by 45% |
| ⚡ 激光炮 (Laser)  | 150 | Extremely fast fire rate |
| 🎯 狙击手 (Sniper) | 130 | Longest range, huge damage |

## 👾 Enemy Types

- **步兵 (Grunt)** — standard infantry
- **快跑者 (Runner)** — fast but fragile
- **重甲 (Tank)** — slow but heavily armoured
- **BOSS** — massive health, appears every 5 waves

## 💡 Tips

- Upgrade towers near chokepoints for maximum efficiency
- Ice towers + Cannon = devastating combo
- Sell under-performing towers and rebuild elsewhere
- Wave bonus gold scales with wave number

## 🛠️ Tech

- Pure HTML5 Canvas rendering
- No frameworks or dependencies
- Mobile touch-friendly
- Runs at 60fps
