# 3DGS 研究报告仓库

这是一个专注于 3D 高斯泼溅（3D Gaussian Splatting，3DGS）技术研究报告的仓库，包含每周更新的论文报告和相关研究资料。

## 仓库结构

- **`3dgs_*.json`** - 每周3DGS论文摘要数据（JSON格式）
- **`3dgs_report*.html`** - 可视化研究报告
- **报告文档** - Markdown格式的技术分析报告

## 每周更新

本仓库每周自动更新以下内容：
1. **`3dgs_YYYYMMDD.json`** - 最新3DGS论文摘要
2. **`3d_ai_YYYYMMDD.json`** - AI相关3D论文摘要
3. **`3d_web_YYYYMMDD.json`** - Web 3D技术论文摘要
4. **可视化报告** - HTML格式交互式报告

- **自动化**：OpenClaw心跳检测 + Cron任务
